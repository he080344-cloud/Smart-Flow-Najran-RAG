# Smart-Flow-Najran-RAG
# 🎓 المساعد الأكاديمي - سمارت فلو (Smart Flow)

مرحباً بك في المستودع الرسمي لمشروع "سمارت فلو"، وهو مساعد أكاديمي ذكي مصمم خصيصاً لطلاب تخصص نظم المعلومات بكلية علوم الحاسب ونظم المعلومات في جامعة نجران. 

يعتمد النظام على تقنية **الاسترجاع المعزز بالتوليد (RAG)** لتقديم إجابات دقيقة وفورية حول الخطط الدراسية، المتطلبات السابقة للمقررات، ولوائح الحذف والإضافة، مما يقلل من الأخطاء الأكاديمية ويوفر وقت الطلاب والمرشدين.

## 🚀 التقنيات والنماذج المستخدمة
تم بناء هذا النظام ليتوافق مع معايير الذكاء الاصطناعي الموثوق (Trustworthy AI)، باستخدام التقنيات التالية:
* **النموذج اللغوي (LLM):** `Qwen2.5-1.5B-Instruct` لتوليد نصوص دقيقة وسريعة تتناسب مع الموارد المتاحة.
* **نموذج التضمين (Embeddings):** `all-MiniLM-L6-v2` لتحويل النصوص إلى متجهات رياضية.
* **قاعدة البيانات المتجهة:** `FAISS` لتخزين واسترجاع القوانين الأكاديمية بكفاءة عالية.
* **واجهة المستخدم:** `Gradio` لتوفير تجربة محادثة تفاعلية وسلسة (تدعم اللغة العربية).

## 🛡️ ميزات النظام (Trustworthy AI)
* **منع الهلوسة:** تم ضبط الأوامر (Prompt Engineering) بدقة لمنع النموذج من تأليف قوانين أو استنتاجات خاطئة، وإجباره على استخراج الإجابات من اللائحة الرسمية فقط.
* **الحماية من الإدخالات الخاطئة:** يتضمن الكود آليات للتعامل مع الإدخالات الفارغة أو غير المنطقية دون انهيار النظام.
* **البيانات المحلية:** النظام مقيد بملف `najran_rules.txt` الذي يحتوي على الخطة الدراسية الفعلية ولوائح الجامعة المعتمدة.
